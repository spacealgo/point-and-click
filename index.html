Rules

A dartboard is moving from the left edge to the right. You should click in the middle of the dartbord. The faster and more accurate your click is the higher scores you get! If you end the game with End button you can continue the game later. You can start a new game with the Start button while the current game is on. If you don't want to continue a saved game just click the Start button twice.
